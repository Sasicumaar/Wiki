To Start Specific App,Navigate to specfic folder (For EX:C:/MPEST/Server) ,then run this below command.

forever start --spinSleepTime 10000 --minUptime 1000 app.js

To Stop Specific App,Navigate to specfic folder (For EX:C:/MPEST/Server) ,then run this below command.

forever stop app.js

To View Running Apps under forever ,then run this below command.

forever list
